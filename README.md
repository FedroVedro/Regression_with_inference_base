# Предсказание цены автомобилей

Домашняя работа №1 по машинному обучению. Задача - предсказать цену автомобиля по его характеристикам.

## Описание проекта

В этом проекте я:
- Провел EDA (разведочный анализ данных)
- Обучил несколько моделей регрессии (Linear, Lasso, ElasticNet, Ridge)
- Создал Streamlit приложение для предсказания цен
- Визуализировал важность признаков

**Лучшая модель:** Ridge Regression с R²=0.6427

## Использование приложения

### Вкладка "EDA"
Здесь можно посмотреть графики:
- Распределение цен
- Зависимость цены от типа топлива
- Scatter plot цены и мощности
- Корреляционная матрица

### Вкладка "Предсказание"
Два способа использования:

**1. Ручной ввод:**
- Вводите характеристики машины (год, пробег, мощность и т.д.)
- Нажимаете кнопку "Предсказать цену"
- Получаете результат

**2. Загрузка CSV:**
- Загружаете файл с данными (можете использовать `example_input.csv` для теста)
- Файл должен содержать столбцы: `year`, `km_driven`, `mileage`, `engine`, `max_power`, `fuel`, `seller_type`, `transmission`, `owner`, `seats`
- Получаете предсказания для всех строк
- Можете скачать результат

### Вкладка "Веса модели"
Показывает какие признаки модель считает важными:
- График топ-15 признаков
- Таблица со всеми весами
- Информация о модели

## Данные

Использовался датасет с ценами на автомобили в Индии.

**Признаки:**
- `year` - год выпуска
- `km_driven` - пробег в километрах
- `mileage` - расход топлива
- `engine` - объем двигателя
- `max_power` - мощность двигателя
- `fuel` - тип топлива (Petrol/Diesel/CNG/LPG/Electric)
- `seller_type` - тип продавца
- `transmission` - коробка передач (Manual/Automatic)
- `owner` - какой по счету владелец
- `seats` - количество мест

**Целевая переменная:** `selling_price` - цена продажи в рупиях

## Результаты

| Модель | Train R² | Test R² |
|--------|----------|---------|
| Linear Regression | 0.5923 | 0.5941 |
| Lasso | 0.5923 | 0.5930 |
| ElasticNet | 0.5672 | 0.5580 |
| **Ridge (с категориальными)** | **0.6597** | **0.6427** |

**Самые важные признаки:**
1. max_power (мощность)
2. year (год)
3. engine (объем двигателя)
